# YouTube Clone

A responsive clone of the YouTube homepage built with **HTML5**, **CSS3**, and **JavaScript (ES6 Modules)**. This project focuses on recreating YouTube's interface while practicing responsive layouts, modern CSS, and dynamic rendering with JavaScript.

## Features

- Responsive YouTube-inspired layout
- Fixed header
- Sidebar navigation
- Search bar with interactive icons
- Custom tooltips
- Notification counter
- Dynamically generated video cards
- Modular JavaScript with ES6 modules
- Clean and organized project structure

## Built With

- HTML5
- CSS3 (Flexbox, CSS Grid, custom properties, pseudo-classes, and pseudo-elements)
- JavaScript (ES6 Modules)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/moussa7elmoussaoui/Youtube-Clone.git
```

### 2. Open the project folder

```bash
cd Youtube-Clone
```

### 3. Run a local development server

> **Note:** This project uses JavaScript ES modules. Opening `index.html` directly with the `file://` protocol will not work because modern browsers block ES modules for security reasons. Please serve the project using a local web server.

For example, using the **Live Server** extension for Visual Studio Code:

1. Install the **Live Server** extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

The project will open at an address similar to:

```
http://127.0.0.1:5500
```
